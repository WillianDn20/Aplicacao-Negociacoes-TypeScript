# AluraBank - Aplicação de Negociações em TypeScript

Aplicação de negociações desenvolvida em TypeScript, demonstrando controle de modelo MVC, uso de decorators, interfaces, classes e DOM. Inclui cadastro e listagem de negociações, regras de validação e interface web com `lite-server`, `concurrently` e compilação com `tsc`.

## Funcionalidades

- Cadastro de negociações com data, quantidade e valor
- Exibição de negociações em tabela
- Uso de classes, interfaces e decorators em TypeScript
- Compilação automática com `tsc -w`

## Como executar

1. Instale as dependências:

   ```bash
   npm install
   ```

2. Compile o projeto:

   ```bash
   npm run compile
   ```

3. Rode o servidor de desenvolvimento e a compilação em watch:

   ```bash
   npm start
   ```

4. Abra o navegador em:

   ```text
   http://localhost:3000
   ```

## Estrutura do projeto

- `app/src/` - código TypeScript da aplicação
- `app/controllers/` - controladores da interface
- `app/models/` - modelos e lógica de domínio
- `app/views/` - renderização e interface do usuário
- `servidor-api/` - servidor de API de exemplo

## Tecnologias

- TypeScript
- lite-server
- concurrently

## Observações

Este projeto serve como base para aprendizado de TypeScript e demonstra padrões de estruturação de código em uma aplicação front-end simples.
